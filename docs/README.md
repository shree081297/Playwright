# Playwright Project Documentation

Quick reference:

- Install dependencies: `npm install`
- Install browsers: `npm run install-browsers`
- Run all tests: `npm test`
- Run headed tests: `npm run test:headed`
- Open HTML report: `npm run show-report`

Tests are located under `test/e2e/spec/` with page objects in `test/e2e/pages/`.

See README.md for full instructions.
